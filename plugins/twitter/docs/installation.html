Installation Instructions

Installation is a very simple process. Just click Install and the plugin installation will be performed.

Getting Twitter Widget ID

First, go to Twitter and create your Twitter Widget. Unfortunately, we do not allow use own settings so you can ignore settings while creating widget.

Once you create your widget, copy long number you see in the URL while editing your Twitter widget.

This number is your Twitter Widget ID, please put it to Admin Panel > Configuration > Miscellaneous > Twitter Widget ID
