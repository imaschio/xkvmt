Installation Instructions

Installation is a very simple process. Just click Install and the plugin installation will be performed.

After the installation you should go to "Admin Panel - Configuration - Financial Configuration" and fill "After Suggest Listing Message" message box (message that will be shown after listing submission with your bank details for the payment).

Also change the email template that will be sent to the user with your bank details for the payment. Here you can edit email templates: "Admin Panel - Email Templates" select "Other Email Templates-Bank account"
