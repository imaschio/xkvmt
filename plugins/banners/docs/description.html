Description

By means of Banner plugin you will be able to:

1. Create/edit/remove banners
2. Choose banner types: html/js, image/flash, plain text, remote
3. Choose banner positions: top/bottom/left/right/user1/user2
4. Specify banner title, alt text, URL, target, banner dimension (in case it is image or flash)
5. Assign banner to definite categories including (or not) subcategories
6. Create banner payment plans and specify its description, cost, days and other paypal options
7. Manage banner payment plans and assign them to layout blocks: top/bottom/left/right/user1/user2
8. Allow banners submission from frontend for website visitors (if at least 1 plan for any banners block created)
